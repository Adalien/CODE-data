# 進貨明細查詢網頁 — 架構說明

> 最後更新：2026-06-10

---

## 三個檔案的分工

| 檔案 | 職責 | 誰來改 |
|------|------|--------|
| `進貨明細_template.html` | HTML 骨架 + 所有 JS/CSS | 要改功能時直接改這個 |
| `gen_html_v2.py` | 讀模板 → 填資料 → 輸出 | 要改資料邏輯時改這個 |
| `進貨明細查詢.html`（輸出）| 完整網頁 | ⚠️ 不要手動改，跑 gen 自動產生 |

---

## 為什麼不會再壞

**以前的問題：**
- Python 用 regex 在 JS 裡找字串替換
- 遇到 `var nm = {}` 就截斷，留下殘碼造成 JS 語法錯誤
- `init()` 失敗 → 0 筆資料、日期空白

**現在的做法：**
- 模板裡只有 `__RAW_DATA_JSON__` 這種佔位符
- Python 只做 `str.replace()`，**完全不碰 JS/CSS**
- 改 JS 就直接改 `進貨明細_template.html`，下次跑 gen 自動套用

---

## 佔位符清單（共 10 個）

| 佔位符 | 填入內容 |
|--------|---------|
| `__RAW_DATA_JSON__` | 進貨明細 Excel 資料（JSON array） |
| `__MAT_NAME_DATA_JSON__` | 原料類品號對照表（JSON object） |
| `__NIMGS__` | 輪播圖片張數 |
| `__SLIDESHOW_IMGS__` | 輪播圖片 base64（進貨明細分頁） |
| `__TAB_TAB_ORDERS__` | 訂單總表頁籤內容 |
| `__TAB_TAB_SHORTAGE__` | 缺料清單頁籤內容 |
| `__TAB_TAB_INVENTORY__` | 庫存現況頁籤內容 |
| `__TAB_TAB_MATERIAL__` | 材料需求頁籤內容 |
| `__TAB_TAB_PICK__` | 須領料頁籤內容 |
| `__TAB_TAB_READY__` | 已備料頁籤內容 |

> BOM_DATA 固定在模板裡（不是佔位符），由缺料分析更新時另行維護。

---

## 使用方式

```
跑「執行缺料分析GUI」
    ↓（跟以前完全一樣）
缺料分析.py → 產生缺料 Excel
gen_html_v2.py → 讀模板 + 填資料 → 輸出 HTML ✅
```

---

## 更新圖片

1. 把圖片放進 `輪播圖片/` 資料夾（支援 jpg/gif/png/webp）
2. 跑一次 gen → 自動讀取，不需改任何程式碼
