## 專案說明
自動化缺料分析腳本，每日執行，比對訂單需求與庫存，輸出缺料清單 Excel。

## 公司資訊
- 公司：達特世生技
- 產品：口罩（QP-707 V10）
- 主程式：`缺料分析.py`
- 執行方式：`執行缺料分析.bat`

## 輸入檔案（自動抓最新檔）
| 檔案 | 說明 |
|------|------|
| 原物料庫存*.xlsx | ERP 匯出的原物料庫存 |
| 耳繩庫存*.xlsx/xls | 耳繩庫存（單位：箱，1箱≈2.777kg） |
| 盒子庫存*.xls | 盒子庫存 |
| 製令總表_合併*.xlsx | 製令 BOM 總表 |
| 訂單生產總表*.xlsx | 訂單需求表 |
| BOM*.xlsx | 主 BOM（Claude紀錄資料夾） |
| INVI02*.XLSX | 品號→品名對照表 |
| 原料類品號*.xlsx | 原料名稱對照表 |

## 輸出
- 缺料分析_MMDD.xlsx（存放於 `缺料分析MMDD/` 資料夾）

## 重要規則（必須遵守）
- 耳繩庫存單位為**箱**，換算：箱數 × 2.777 = kg
- 3D 袋裝口罩外層布用量需套用 3D 幾何公式（非平面計算）
- XL 尺寸外層布片數：630片/kg
- L 尺寸：680片/kg、M：730片/kg、S：790片/kg
- 盒裝與袋裝耗材不同，需分開計算

## 品號結構
- BPP 開頭：外層布
- BMB 開頭：熔噴布
- BES 開頭：親膚布
- BEL 開頭：耳繩
